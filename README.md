# 📊 Download-Zero (下载零)

> **本应用完全由 AI (Manus) 根据用户需求自主编写完成。**

`Download-Zero` 是一个基于 Alpine Linux 的极简、高效流量消耗工具。它专为网络压力测试或流量配额消耗场景设计，具备高度的自动化和智能化特性。

---

## 🌟 核心作用

*   **自动化流量消耗**：通过轮询下载全球高速测速节点或大厂系统镜像文件来产生流量。
*   **零磁盘损伤**：采用数据流重定向技术，下载数据直接丢弃到 `/dev/null`，**完全不占用硬盘空间**，对 SSD/HDD 无任何读写损伤。
*   **极致轻量**：基于 Alpine Linux 镜像，容器运行内存占用极低（通常小于 64MB）。
*   **智能推送**：每日任务完成后，自动通过企业微信 Webhook 发送**人性化纯文本报告**，兼容微信端查看。

---

## 🛠️ 主要特性

*   **双重随机化**：每个下载循环的流量大小（如 1-5GB）和休息时间（如 10-30分钟）均在设定范围内随机波动，模拟真实用户行为。
*   **严格限额与限速**：
    *   支持 24 小时总流量上限，达到上限后自动进入休眠，次日凌晨自动唤醒继续。
    *   支持自定义下载限速，防止占满带宽影响其他业务。
*   **高可用故障转移**：内置 9 个下载源。若某个源连续 3 次下载失败，将自动禁用该源并记录到日志中，确保任务不中断。
*   **人性化统计**：
    *   自动将秒数转换为“小时/分/秒”格式。
    *   统计今日总流量、各下载源贡献流量、平均速度及失效源编号。

## 🚀 快速使用方法

### 1. 准备环境
确保您的设备已安装 `Docker` 和 `Docker Compose`。

### 2. 获取项目文件
将以下文件放在同一个文件夹中：
*   `docker-compose.yml`
*   `run.sh`
*   `.env`
*   `README.md`

### 3. 配置参数
编辑 `.env` 文件，根据您的需求修改参数。**请务必填入您的企业微信 Webhook 地址**。

### 4. 一键启动
在文件夹内打开终端，执行：
```bash
docker compose up -d
```

## ⚙️ 配置参数说明 (.env)

| 变量名 | 默认值 | 说明 |
| :--- | :--- | :--- |
| `DAILY_LIMIT_GB` | `150` | 24小时内最大下载总量 (GB) |
| `SPEED_LIMIT` | `10M` | 最大下载速度限制 (如 10M, 500K) |
| `MIN_LOOP_GB` / `MAX_LOOP_GB` | `1` / `5` | 单次循环下载流量随机范围 (GB) |
| `MIN_SLEEP_MINUTES` / `MAX_SLEEP_MINUTES` | `10` / `30` | 单次循环后休息时间随机范围 (分钟) |
| `WECHAT_WEBHOOK` | `(必填)` | 企业微信 Webhook 推送地址 |
| `URLS` | `(9个URL)` | 下载源地址，用英文逗号分隔 |
